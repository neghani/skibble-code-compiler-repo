In this example, the header is fixed to the top of the page using the position property with a value of fixed. The sidebar is positioned using the position property with a value of fixed as well, but its position is relative to the top of the page and to the left side of the viewport. The content area is positioned with a margin-left that clears the space used by the sidebar. The use of the flexbox layout in the container element allows the sidebar and the content area to be displayed side by side.
